Ubuntu 14.04 编译wireshark source

由于通过apt-get install wireshark安装的wireshark版本比较旧，不支持BLE协议。所以，得下载源代码进行编译。

1.下载wireshark的源代码 2.安装必要的依赖库sudo apt-get build-dep wireshark 3.

cd wireshark
./configure
make -j8
make install

然而，大部分情况下，这样是会有错误的。尤其是在./configure后，会报缺少一些库，这个时候可以根据缺少的库，一个个的安装。

我的情况下是缺少qt相关的库，google后，还是不能解决：qt is not found 或者 can't find uci. 可以参考：qt is not found，Qt not available error? 我有尝试apt-get install qt5-default和apt-get install qt4-default，最后依然报错。

最终，看到了这个使用GTK安装wireshark，安装qt会有各种错误，使用这个后，很顺利的安装成功了。
